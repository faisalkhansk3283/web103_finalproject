# User Stories

## User Roles
- **User** — a person tracking their personal income and expenses

## Stories

1. As a **user**, I want to view all my transactions in one place, so that I can see my overall spending at a glance.
2. As a **user**, I want to add a new transaction, so that I can keep my records up to date.
3. As a **user**, I want to edit an existing transaction, so that I can fix mistakes like a wrong amount or date.
4. As a **user**, I want to delete a transaction, so that I can remove entries that were added by mistake.
5. As a **user**, I want to assign one or more categories to a transaction, so that I can organize my spending by type.
6. As a **new user**, I want default categories to be created automatically when I sign up, so that I don't have to set everything up manually.
7. As a **user**, I want to add a transaction through a quick modal, so that I don't have to leave my dashboard to log an expense.
8. As a **user**, I want the app to prevent me from saving a transaction with a negative amount, so that my data stays accurate.
9. As a **user**, I want the app to require a category before saving a transaction, so that nothing goes untracked.
10. As a **user**, I want to filter my transactions by category, so that I can see how much I've spent in a specific area.
11. As a **user**, I want to sort my transactions by date, so that I can quickly find my most recent spending.
12. As a **user**, I want to see a clear error message if my transaction submission fails, so that I know what went wrong and how to fix it.
